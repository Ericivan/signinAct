#####b1.1签到接口

> api/sign

> 请求方式：POST

> 接口说明：签到活动进行签到的接口

参数说明:

| 参数名 | 是否必须（Y/N) | 类型 | 参数说明 |
| ------ | -------------- | ---- | -------- |
|        |                |      |          |

返回结果：

```json
{
    "message":"succeed",
    "code":0
}
```



错误码返回

| 错误码 | 错误码说明           |      |
| ------ | -------------------- | ---- |
| 500    | 系统错误             |      |
| 1000   | 用户当天已经进行签到 |      |

#####1.3 补签接口接口

> api/sign/re

> 请求方式：POST

> 接口说明：用户完成补签任务后进行补签

参数说明:

| 参数名 | 是否必须（Y/N) | 类型 | 参数说明 |
| ------ | -------------- | ---- | -------- |
|  date      |         Y       | 日期时间 | 补签日期，eg:'2018-01-01' |

返回结果：

```json
{
   "message":"succeed",
    "code":0
}
```



错误码返回

| 错误码 | 错误码说明           |
| ------ | -------------------- |
| 500    | 系统错误             |
| 10000  | 用户当天已经进行签到 |
|  10001      |  用户当天已经进行补签                |
| 10002    |             传入补签时间有误|
